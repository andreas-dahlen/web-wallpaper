# ADWebWallpaper

A WebView-based live wallpaper for Android 15.  
Includes HTML/CSS/JS assets, Kotlin code for WebView wallpaper service.

## Features
- Live WebView wallpaper
- Interactive buttons to launch apps via JS
- Full-screen HTML layout
- Placeholder clock/animations

## Structure
- `app/src/main/assets/` → HTML/CSS/JS
- `app/src/main/java/com/ad/webwallpaper/` → Kotlin files
- `res/xml/wallpaper.xml` → wallpaper declaration
